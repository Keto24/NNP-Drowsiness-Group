Driver Drowsiness Detection

By Yelnur Abdushev, Emma Gandonou, Keto

Outline and Related Works

Introduction

Drowsy driving is a serious issue, comparable to driving under the influence. This project aims to develop a neural network capable of detecting driver drowsiness in real time.

While existing technologies address drowsy driving, our system is unique in its specialization for individuals wearing glasses. We integrate OpenCV, TensorFlow, and Twilio API. The model is trained on multiple datasets, including OC oC Drowsiness - Detection, MRL Eye Dataset, and Driver Drowsiness Dataset. These diverse data sources reduce bias and improve accuracy, ensuring coverage for edge cases.

Thesis

The driver drowsiness detection system will help save thousands of lives. AI can be used to improve road safety.

Method

  • Facial feature extraction for detecting drowsiness signs.
  • Integration of machine learning models with real-time alerts.
  • Testing across diverse environments to ensure generalizability.

Ethical Considerations

Ethical concerns include privacy risks related to camera usage and the consequences of alerting law enforcement. Bias in detection methodologies is also considered, with mitigation strategies explored.

Conclusion

Our drowsiness detection system will significantly improve road safety, potentially preventing thousands of accidents.

Future Work

We aim to expand the system for integration into built-in vehicle safety mechanisms.

Related Works

Existing drowsiness detection systems rely on CNN or ANN models. Salem & Waleed (2024) developed a real-time CNN-based model achieving 90-99.86% accuracy across multiple datasets [1]. Their system integrates MobileNetV2, InceptionV3, and Haar Cascade for feature extraction.

Inkeaw et al. (2022) proposed an ANN-based system using visual facial descriptors like EAR, MAR, face length (FL), and face width balance (FWB) [2]. The system integrates EEG-based microsleep detection and facial feature analysis.

Albadawi et al. (2022) reviewed drowsiness detection methodologies, categorizing them into physiological, behavioral, and vehicle-based approaches [3]. They highlighted challenges related to real-time implementation and sensor reliability.

Commercial solutions such as Motive’s Drowsiness AI and Bosch’s Driver Drowsiness Detection utilize AI and sensor-based monitoring [4][5]. These solutions monitor steering behavior, lane deviation, and facial expressions to detect drowsiness.

Chinthalachervu et al. (2022) implemented a real-time drowsiness detection system using webcam-based facial expression analysis [6]. The model calculates EAR, mouth opening ratio, and nose length ratio to detect fatigue.
